Fish Annotator — Help

What it is for

Fish Annotator lets you draw and adjust outlines (“masks”) of anatomical features on zebrafish images — body contour, eye, notochord, and so on. It saves them as a FishInspector __SHAPES.json file for analysis in FishInspector. Use it two ways: annotate an image from scratch, or load an existing mask and correct it.

1 · Prepare images: TIFF → JPG

Microscopes usually save TIFF. Fish Annotator runs fastest on JPG, so convert first:

  • Save as JPG at 90% quality or higher.
  • Keep the exact same pixel size as the original — essential, because the annotation coordinates must line up with the original for FishInspector.
  • Keep your original TIFFs; the JPG is only for annotating.

Conversion runs on a computer — see Converting TIFF → JPG below.

2 · Keep each image and its JSON together

Put each image and its annotation file <image name>__SHAPES.json in the same folder. When you open them together, Fish Annotator pairs them automatically by name and shows the existing outlines. The matching is tolerant of spaces, underscores, and .ome/.tif differences.

3 · Open & save — by device

Open with Load images + JSON (select the image and its JSON in one dialog) or Load folder (loads every image + JSON in a folder, where supported). Save active JSON writes <image>__SHAPES.json.

On most devices the saved file lands in Downloads — the app cannot write back into a cloud folder — so you then move it into your image folder. (Exception: Android Chrome / desktop can save into a chosen folder.)

4 · The tools

  • Draw — trace a feature’s outline with the pen. One outline per feature; drawing again replaces it.
  • Edit — push an existing outline into place. A circle (the Brush) follows the pen and shoves the line out of the circle, away from the brush — like a disc pushing a string. Push from one side to move the line that way; approach from the other side to push it back. The Brush slider (1/100–1/3 of image height) sets how much of the line moves at once. On a computer the circle follows the mouse so you can see where it will act, and it pushes only while a button is held; with a pen it acts only while the pen touches the screen.
  • Erase — a brush (size set by the Brush slider) that removes only the parts of the active feature’s outline it touches. Use Clear to remove a whole feature at once.
  • Pan / zoom — one finger pans, two fingers pinch to zoom; the pen always draws. Also Fit, −, 100%, +.
  • Image view (Brightness / Contrast) — sliders in the sidebar adjust how the image looks on screen so you can see faint edges. They do not change the image file. Tap reset to return to normal.
  • All features — shows every outline at once for review (editing paused). Tap a feature to resume editing it.
  • Feature tiles — choose the feature you’re working on. The dot is red when a feature has no outline and green once it does — a quick check that nothing is missing.
  • Undo, Clear (removes the current feature’s outline), and Save active JSON. Unsaved JSON shows in yellow when you have changes not yet saved.

5 · Typical workflow

Load the image (and its JSON if it exists) → pick a feature → trace it, or use Edit to adjust → check every dot is green → Save active JSON → move the JSON into the folder with the original TIFF → open in FishInspector for analysis.

Converting TIFF → JPG (on a computer)

Do this on a PC/Mac, not the tablet. It keeps the JPG’s pixel dimensions identical to the TIFF (required for FishInspector). The drawing tool itself needs no installation — just open index.html in a browser. There is no in-page converter yet; use the convert_to_jpg.py script:

  1. Install Python 3 from python.org. On Windows, tick “Add Python to PATH” during setup.
  2. Open a terminal / Command Prompt and install the libraries:
    pip install pillow tifffile numpy
  3. Run the converter:
    python convert_to_jpg.py INPUT_FOLDER OUTPUT_FOLDER

Defaults: JPG quality 90, grayscale, same pixel size. That is the entire setup.
